# 🏦 Sistema de Login en C#

Este proyecto es una aplicación de consola desarrollada en C# que simula un sistema de autenticación de usuarios con control de intentos.

---

## 🚀 Funcionalidades

* ✅ Validación de usuario y contraseña
* ✅ Manejo de múltiples usuarios con `List<>`
* ✅ Control de intentos (máximo 3)
* ✅ Bloqueo automático tras intentos fallidos
* ✅ Mensajes dinámicos al usuario

---

## 🧠 Lógica implementada

El sistema utiliza:

* Ciclos `while` para controlar intentos
* Ciclos `for` para recorrer usuarios
* Listas (`List<string>`) para almacenar datos
* Condicionales (`if / else`) para validación

---

## 💻 Tecnologías utilizadas

* C#
* .NET
* Consola (CLI)

---

## ▶️ Cómo ejecutar el proyecto

1. Clonar el repositorio:

```bash
git clone https://github.com/TU-USUARIO/LoginSystem-CS.git
```

2. Entrar a la carpeta:

```bash
cd LoginSystem-CS
```

3. Ejecutar:

```bash
dotnet run
```
